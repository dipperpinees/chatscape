## Install package
### `npm install && npm install --prefix client` 
or
###  `yarn install && yarn install --prefix client`

## How to start
### `npm start` or `yarn start`
Open [http://localhost:3000](http://localhost:3000) to view it in the browser (server running on port 8000).
